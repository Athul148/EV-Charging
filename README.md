# 🚀 EV Charging Dashboard - Power BI

## 📊 Project Overview
This project analyzes EV charging station usage, vehicle trends, and demand patterns.

## 🛠 Tools Used
- Power BI
- DAX
- Excel

## 📈 Key Insights
- Peak usage during evening hours
- Certain stations have high demand
- Vehicle type impacts usage patterns

## 📂 Files Included
- PowerBI Dashboard (.pbix)
- Dataset
- Report (PDF)
- Screenshots
